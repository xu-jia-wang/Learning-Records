linux 常用命令

命令 作用
ls(list) 查看当前目录下的内容
pwd 查看当前所在的目录
cd[目录名] 切换目录
touch[文件名] 如果文件不存在，新建文件
mkdir[目录名] 创建目录
rm[文件名] （remove） 删除指定文件

技巧：

  • Tab 自动补全
  • 两次 tab 给出操作提示
  • 使用ctrl+l快捷清屏

Linux 命令格式：

command [-options] [parameter]
command: 命令名
options: 选项，可以用来对命令进行控制，也可以省略
parameter: 传给命令的参数，可以为0，1也可以为多个

文件目录操作命令

  • ls [-al] [dir] 显示指定目录下的内容(使用 ll代替 ls -l)。
  • cd [dirName] ~:当前用户的home目录，.:表示目前所在的目录，..:表示目前目录位置的上级目录。
  • cat [-n] filename 用于显示文件内容 -n:由1开始对所有输出的行数编号。
  • more filename 以分页形式显示文件内容，回车：向下一行，b:回退上一屏，q:退出，空格：下一屏。
  • tail [-f] filename 查看文件末尾内容，-f:动态的读取文件末尾内容并显示，用于日志文件的内容输出。
  • echo 输入内容 >> 追加的文件 ，其中如果是>>那么追加到文件中去，>表示覆盖。
  • mkdir [-p] dirname 创建目录 -p:确保目录存在，不存在就创建一个，可以实现多层目录同时创建。
  • rmdir [-p] dirname 删除空目录,-p:子目录被删除后使得父目录为空，则一并删除，rmdir itcast*,删除以itcast开始的空目录
  • rm [-rf] name 删除文件或者目录，-r:将目录及目录中所有文件逐一删除，-f:无需确认，直接删除
  • cp [-r] source dest 复制文件或目录，-r:如果复制的是目录，将复制该目录下的所有子目录和文件 可以选择将整个目录复制到另外目录下去，也可以选择将目录下的内容都复制进去，例如：cp -r itcast/* itheima/
  • mv source dest 为文件或者目录改名，或将文件或者目录移动到其它位置

tips:
-a:显示所有文件及目录包括隐藏文件
-l:显示更多的详细信息

打包压缩命令

tar [-zcxvf] filename [files]
包文件后缀为.tar表示只是完成了打包，并没有压缩
包文件后缀为.tar.gz表示打包的同时还进行了压缩
说明：

  • -z: z代表gzip,通过gzip命令处理文件，gzip可以对文件压缩或者解压
  • -c: c代表create,创建新的包文件
  • -x: x代表的是extract,实现从包文件中还原文件
  • -v: v代表的是verbose,显示命令执行过程
  • -f: f代表 file,用于指定包文件名称
    组合：xvf 解包,zxvf 解压并解包，cvf 打包 , zcvf 打包并压缩
    tar -zxvf hello.tar.gz -C /usr/local，将文件解压到指定的目录去

文本编辑命令 Vim

作用： 对文件内容进行编辑，vim实际上是一个文本编辑器 vim fileName
说明：

  1. 在使用vim命令编辑文件时，如果指定文件不存在则创建文件。
  2. 分为三种模式，分别为：命令模式(Command mode),插入模式(Insert mode),底行模式(Last line mode).这三种模式之间可以相互切换。
    1. 命令模式：
      • 可以查看文件内容，移动光标(上下左右箭头,gg(开头),G(末尾))
      • 通过vim命令打开文件后，默认进入命令模式
      • 另外两种模式需要首先进入命令模式之后，才能进入彼此
    2. 插入模式
      • 插入模式下可以对文件内容进行编辑
      • 在命令模式下按下[i,a,o]其中一个就可以进入
      • esc回到命令模式
    3. 底行模式
      • 底行模式可以对文件内容进行查找，显示行号，退出等操作
      • 在命令模式下按下[:,/]中一个，进入底行
      • 通过/进入底行模式，可以对文件进行查找
      • 通过:进入底行模式，可以输入wq(保存并退出),q!(不保存退出), set nu(显示行号)

查找命令

  1. 在指定目录下查找文件
    find dirName -option fileName,其中option 一般是指定名称-name
    例如 find . -name “*.java” 在当前目录下按照名称寻找以.java 结尾的文件
  2. grep word fileName 从指定的文件中查找文本内容。
    例如：grep Hello *.java
